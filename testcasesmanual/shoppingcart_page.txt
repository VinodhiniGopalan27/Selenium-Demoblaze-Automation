
Test Scenarios for Adding items in the Shopping Cart:
1.	Verify that on the home page, the user can click the link to the desired product (i.e., Nokia Lumia 1520) and view the product description and price.
2.	Check whether the Cart button is seen on the product display page.
3.	Verify the Add button is clickable or not.
4.	Verify that the user can add one or more products to the cart.
5.	Verify that that user sees the success message “Product added” when clicking the add button.
6.	Check on adding the new products to the cart, the count is increasing.
7.	Verify whether the same item can be added multiple times to the cart.
8.	Verify that the user can see the previously added products in the cart page every time he signs in into Demoblaze.
9.	Check whether the page redirects to the product detail page when clicking on the product in the shopping cart.
10.	Verify that the max limit to the number of products a user can add into the cart.
11.	Verify that the user can change the quantity of the added items.
12.	Verify that the user can check the sequence order of the items as per the items added to the cart.

Test Scenarios for Deleting items in the Shopping Cart:
1.	Verify that the delete link displayed in the cart window after every product.
2.	Verify the delete link is functioning as per the requirement.
3.	Check that the user is allowed to delete one item from the cart
4.	Check that the user is allowed to delete multiple items from the cart.
5.	Verify that the user can view the other products in the cart after successful deletion.
6.	Verify that the user can check the sequence order of the items as per the items deleted from the cart.
